# Character Generation Prompts — Limonchero 3D
**Uso:** Copiar el prompt completo en Midjourney, DALL-E 3, o Stable Diffusion  
**Estilo base de referencia:** concept sheet de Limonchero (low-poly facetado, cartoon-noir, años 50)  
**Fecha:** 2026-04-16

---

## Instrucciones de Uso

Cada prompt incluye:
- **Prompt principal** — pegar tal cual en la herramienta
- **Parámetros Midjourney** — agregar al final si usas MJ
- **Notas de ajuste** — qué cambiar si el resultado no es el esperado

El prompt base de estilo se repite en todos los personajes para mantener coherencia visual. **No lo elimines.**

### Limoncito
*Asistente del detective. Key lime pequeño y nervioso.*

```
low-poly faceted 3D character, anthropomorphic key lime fruit character, 
film noir 1950s aesthetic, hard side lighting, dark moody background, 
cartoon stylized proportions, angular geometry with visible polygon faces, 
flat color areas, detective mystery atmosphere, character sheet white background,
bright lime green skin color #8BC34A, small round head larger than body proportions, 
short stature energetic pose, no hat, fitted clothing, 
eager nervous expression wide eyes, 1950s assistant outfit vest and shirt, 
full body front and side view, character turnaround sheet
```

**Parámetros Midjourney:** `--ar 3:4 --style raw --v 6`

**Notas de ajuste:**
- Si sale demasiado realista → agregar `extremely stylized, non-realistic, toon shader`
- Si la piel no es verde lima → agregar `bright lime green skin, NOT yellow`
- Si sale muy alto → agregar `short character, small stature, 75% height of average`

## Notas de Consistencia entre Personajes

Al generar en distintas sesiones, agregar estas frases para mantener coherencia:

- `same art style as [referencia anterior]` — si Midjourney tiene referencia de imagen
- `consistent low-poly style, same polygon density across all characters`
- `noir 1950s color grading, desaturated background, characters as main color source`

Si algún personaje sale con estilo muy diferente al resto, regenerar usando como `--sref` (Midjourney style reference) la imagen del personaje que salió mejor.
